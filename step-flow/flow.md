## 全体フロー

```
[LINE友だち登録]
      │
      ▼
 ① LINE #1（登録直後）
   あいさつ ＋ メルマガ登録を促す
   ※登録経路がわかるタグ設定
      │
      ▼
 ② メルマガ #2（登録直後）
   あいさつ ＋ 特典配布 ＋ 価値提供
   ※メルマガ登録者にラベル付け
      │
      ├─── メルマガ未登録の場合（30分後）
      │         ▼
      │    ③ LINE #3（登録30分後 ＋ メルマガ登録ラベルなし）
      │      メルマガ登録と特典受取を再度促す
      │
      ▼
 ④ メルマガ #4（登録1日後 18:00）
   特典を受け取っているか再確認
```
